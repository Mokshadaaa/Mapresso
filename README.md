# ☕ Mapresso

Mapresso is a location-based cafe discovery platform that helps users find nearby cafes, view them on an interactive map, and discover places based on distance and ambience.

## 🚀 Features

- 📍 Detects user's current location
- 🗺️ Interactive map using OpenStreetMap
- ☕ Finds nearby cafes
- 📏 Distance calculation
- 🔍 Cafe search functionality
- 🎭 Mood-based cafe tags
- 📸 Aesthetic score system

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- OpenStreetMap
- React Leaflet
- Overpass API
